# EdgeFinder Pro — Deploy to Vercel (Free)
# Takes about 10 minutes total. Follow each step exactly.

═══════════════════════════════════════════════════════
STEP 1 — Create a GitHub account (if you don't have one)
═══════════════════════════════════════════════════════
1. Go to https://github.com
2. Click "Sign up"
3. Enter email, password, username
4. Verify your email

═══════════════════════════════════════════════════════
STEP 2 — Create a new GitHub repository
═══════════════════════════════════════════════════════
1. Click the "+" icon top-right → "New repository"
2. Name it: edgefinder-pro
3. Set to Private (recommended)
4. Click "Create repository"
5. You'll see a page with a URL like: https://github.com/YOURNAME/edgefinder-pro

═══════════════════════════════════════════════════════
STEP 3 — Upload the project files
═══════════════════════════════════════════════════════
On the new repo page, click "uploading an existing file"

Upload ALL files from the edgefinder-app folder, keeping this structure:
  pages/
    index.js
    api/
      odds.js
      claude.js
      injuries.js
  package.json
  next.config.js

DO NOT upload .env.local (that contains your secret keys — add them in Step 5)

Click "Commit changes"

═══════════════════════════════════════════════════════
STEP 4 — Create a Vercel account & connect GitHub
═══════════════════════════════════════════════════════
1. Go to https://vercel.com
2. Click "Sign Up" → choose "Continue with GitHub"
3. Authorize Vercel to access your GitHub
4. Click "Add New Project"
5. Find "edgefinder-pro" in the list → click "Import"
6. Framework Preset will auto-detect as "Next.js" ✓
7. DO NOT click Deploy yet — do Step 5 first

═══════════════════════════════════════════════════════
STEP 5 — Add your secret API keys to Vercel
═══════════════════════════════════════════════════════
On the import page, expand "Environment Variables" and add:

  Name: ODDS_API_KEY
  Value: cd69f10ac2a8324c03c26303c5649271

  Name: ANTHROPIC_API_KEY
  Value: [your Anthropic key — get free at console.anthropic.com]

Click "Add" after each one.

═══════════════════════════════════════════════════════
STEP 6 — Deploy!
═══════════════════════════════════════════════════════
1. Click "Deploy"
2. Wait ~2 minutes for the build to finish
3. You'll see confetti and a URL like: https://edgefinder-pro.vercel.app

That's your app! Share the link with anyone.

═══════════════════════════════════════════════════════
SHARING
═══════════════════════════════════════════════════════
- Anyone with the link can use it — read only, no editing
- Works on phone, tablet, desktop, any browser
- No login required for users
- You can change the URL to something custom in Vercel settings

═══════════════════════════════════════════════════════
UPDATING THE APP IN THE FUTURE
═══════════════════════════════════════════════════════
Just replace files in your GitHub repo → Vercel auto-redeploys in ~1 min

═══════════════════════════════════════════════════════
COSTS
═══════════════════════════════════════════════════════

- GitHub: FREE
